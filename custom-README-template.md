## What I learned
I learned how to create a responsive QR code card using Flexbox and media queries. I practiced organizing HTML and CSS clearly with descriptive class names. I also improved my understanding of mobile-first design and centering content vertically and horizontally.

## Challenges encountered
The main challenge was keeping the card layout centered on both mobile and desktop. Another challenge was placing the attribution text below the card while keeping it centered. I solved these issues using `flex-direction: column` and media queries.

## Improvements for future projects
Next time, I would focus more on semantic HTML and accessibility, like using `<figure>` and `<figcaption>`. I would also add subtle animations or transitions to make the component more interactive. Finally, I would refine CSS organization for even cleaner, maintainable code.
